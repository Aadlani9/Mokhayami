مخيمي (Mokhayami)
مخيمي هو تطبيق مبدئي يهدف إلى تسهيل عملية تأطير وتنظيم المخيمات الصيفية. الهدف من هذا المشروع هو تجربة فكرة لتطوير أداة تساعد المؤطرين في إدارة الأنشطة، متابعة المشاركين، والتواصل بشكل أكثر فاعلية. يرجى ملاحظة أن هذا ليس هو الإصدار النهائي للتطبيق بل هو مجرد تجربة أولية.

💡 المميزات (قيد التجربة)
إدارة وتنظيم الأنشطة داخل المخيم.

متابعة المشاركين وتحديث المعلومات.

أدوات تواصل مع المشاركين وأولياء الأمور.

واجهة بسيطة وسهلة الاستخدام.

تجربة أولية يمكن تطويرها بشكل أكبر بناءً على الآراء والتعليقات.

🛠 التكنولوجيا المستخدمة
HTML: لبناء الهيكل الأساسي للتطبيق.

CSS: لتنسيق وتصميم واجهات المستخدم.

JavaScript: لإضافة التفاعلية (إذا تم استخدامه).

PHP (اختياري): في حالة استخدام قاعدة بيانات أو واجهات خلفية.
