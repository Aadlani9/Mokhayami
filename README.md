# مخيمي (Mokhayami)

**مخيمي** هو تطبيق موجه لتسهيل عملية تأطير وتنظيم المخيمات الصيفية. يهدف المشروع إلى توفير أدوات رقمية تسهل على المؤطرين إدارة الأنشطة ومتابعة المشاركين بشكل فعال.

---

## 💡 **المميزات**

- إدارة وتنظيم الأنشطة داخل المخيم.
- متابعة المشاركين وجدولة الأنشطة.
- أدوات للتواصل مع المشاركين وأولياء الأمور.
- واجهة بسيطة وسهلة الاستخدام.
- تطبيق موجه للمخيمات الصيفية في المغرب.

---

## 🛠 **التكنولوجيا المستخدمة**

- **HTML**: لبناء الهيكل الأساسي للتطبيق.
- **CSS**: لتنسيق وتصميم واجهات المستخدم.
- **JavaScript**: لإضافة التفاعلية (إذا تم استخدامه).
- **PHP** (اختياري): في حالة استخدام قاعدة بيانات أو واجهات خلفية.

---

## 🚀 **التثبيت**

1. قم بتحميل المشروع إلى جهازك:
   ```bash
   git clone https://github.com/yourusername/mokhayami.git
   ```

2. افتح المشروع باستخدام محرر النصوص المفضل لديك (مثل Visual Studio Code).

3. إذا كنت بحاجة إلى تشغيل التطبيق محليًا باستخدام **خادم محلي** (مثال: XAMPP أو WAMP):
   - ضع ملفات المشروع داخل مجلد الخادم.
   - افتح المتصفح وتوجه إلى `localhost/yourprojectfolder`.
